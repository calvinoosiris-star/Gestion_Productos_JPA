# Gestion de Productos _ JPA
Mini proyecto CRUD desarrollado en Java utilizando JPA (Hibernate) y MySQL.

# Funcionalidades
-Crear producto
-Listar producto
-Eliminar producto

## Tecnologias utilizadas
-Java
-JPA
-Hibernate
-MySQL
-Maven

## Estructura
Proyecto Maven estructurado en capas utilizando persistencia con JPA.
